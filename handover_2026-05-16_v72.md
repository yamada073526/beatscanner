## 3. 触ると危険な箇所 (v72 で確定した永続原則)

### v72 新規確定原則

**lightweight-charts v5 migration** → `feedback_pane3_detail_view.md` Phase 3-d round 8 / silent fail hook
- `ISeriesApi.setMarkers()` は v5.2.0 で削除済、 `createSeriesMarkers(series, markers)` 必須
- `.claude/hooks/pre_edit_lightweight_charts.sh` で grep block (Phase 3-a / 3-c で同 root cause で 2 回 silent fail 教訓)
- chart marker は `markersRef.current.setMarkers()` で incremental update (chart 全焼却回避)

**emoji を canvas に焼かない (Aman 級維持)** → `feedback_pane3_detail_view.md` Phase 3-d round 7
- lightweight-charts text 描画は OS 依存の native emoji glyph を低解像度 raster で焼く → 品格欠如
- 解: chart canvas は dot only (text: '')、 意味伝達は HTML 凡例 (Lucide SVG icon) + ticker badge

**TickerBadge primitive SSOT** → `chip_primitive_canonical.md` 拡張候補
- `frontend/src/components/ui/TickerBadge.jsx` (logo + ticker SSOT)
- 4 体合議で確定、 size preset xs=14 / sm=20 / md=28 / lg=56
- 既存 9 callsite (App / Judgment / Profile / Pane 4 NewsItem / TxHistory) は将来別 PR で機械置換予定

**バックテスト forex は固定レート (Brinson 帰属分析)** → `project_backtest_phase1_design.md`
- USD/JPY 150 円固定、 「5 条件の α」 と「円安 β」 の交絡防止
- forex 込み実 P/L は Phase 2.5+ で opt-in toggle (上級者向け) として追加候補

- **期間 5 年** (Starter limit、 Phase 3 で Premium upgrade 検討)
