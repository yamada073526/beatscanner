## 3. 🟡 重要知見 (v51 セッション固有)

### 3-1. 「動かないが状態が変わる」は最優先原則 (Linear / Aman / Apple)
- レイアウトシフトを起こす UI 変更は **Trust Cliff の最重要バグカテゴリ**
- 状態変化は色 / 縁取り / 数字書換 / chip 出現で表現し、高さや幅を変えない
- 例外: ユーザーが明示的にクリックして展開する accordion 等は OK

### 3-2. framer-motion (44KB) は不採用、CSS-only or motion-one (5KB)
- 3 体合意: framer-motion は ROI 悪い
- KPI count-up は自前 RAF (~30 行、+0.5KB) で十分滑らか
- 必要時のみ motion-one を `manualChunks: 'luxury'` で分離 + lazy load

### 3-3. デザイントークン命名は「Stripe Dashboard + Linear + Vercel Geist」 ハイブリッド
- spacing: `--space-1 / 2 / 3 / 4 / 6 / 8 / 12 / 16` (4-base scale)
- radius: `--radius-xs / sm / md / lg / xl / pill`
- shadow: `--shadow-1 / 2 / 3 / 4` + `--shadow-glow-cyan` (Stripe 流階層)
- motion: `--ease-out-expo / --ease-in-out-quad`, `--motion-fast / base / slow / stage`
- z-index: `--z-base / sticky / overlay / modal / toast`

### 3-4. `prefers-reduced-motion` はグローバル CSS で 1 ルール一括無効化
```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```

### 3-5. CLS=0 を維持するための徹底事項
- `tabular-nums` を全数値表示に適用 (桁ズレ防止)
- 画像は `aspect-ratio: 1` 強制 (CompanyLogo 等)
- table tbody は `content-visibility: auto + contain-intrinsic-size: 0 44px` で遅延描画 (LCP 改善)
- 状態変化で挿入される DOM は `min-height` で空間予約 OR title row 等の既存スロットに吸収
