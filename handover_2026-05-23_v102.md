## 4. 触ると危険な箇所 (v101 + v102 教訓)

### v101 から継承
- sticky 検索バー / 発光系 / VITE_ARG/ENV 同期 / Hallucination Guard / FMP /stable 移行 / vision-eval frame 8 / ProfileCard sanitize

### v102 新規教訓
- **Pane 4 ReadingMode 全面 Overlay 化 (B-E) で旧 PanelGroup 55/45 縦割を破棄** — `react-resizable-panels` import を Pane4Inspector から削除。 close→list の 1-click ナビが Linear/Gmail 流 (UX 変更なので user dogfood feedback 受け次第 PanelGroup 復帰判断あり得る)
- **NewsItem の `ws-pane4-news-item` CSS は v66 §1 round 3 で `backwards` 戦略確立済、 framer-motion で動作を重ねると hover/active transform 保護を壊す risk** — B-motion で意図的に CSS slide-in を keep、 framer-motion は ReadingMode のみ
- **B-L で title 2 行 → 1 行 line-clamp 変更** — タイトルが切れて Reading Mode 開かないと全文見えない設計。 dogfood で「タイトル切れる」 と criticism 来たら 2 行 + fontSize 11px に pivot 候補
- **Sprint B-pro-badge は subscription state 受け取らず常時表示**。 Pro user に redundant だが「Pro 機能訴求」 として Stripe/Notion 流。 不要なら Pane4Inspector の `usePane4Section` 等から `isProUser` を受け取って隠す改修可能 (0.3 人日)
