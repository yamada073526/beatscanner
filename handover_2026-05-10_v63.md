## 7. 🟡 触ると危険な箇所

### v63 追加
- **`useArrivalSpotlight.js` (band-based v62)**: 単一スポットライト → 帯ベースに変更済。CSS 触らず JS 選定ロジックのみ修正で安全。`memory/glow_elevation_postmortem.md` v62 章追加済
- **`workspaceStore.js` (Zustand persist)**: `bs:ws:store:v1` キー、partialize で URL state を除外。新 state 追加時は partialize 更新を忘れない
- **`useUrlSync.js` の replaceState**: pushState は履歴汚染、絶対採用しない
- **`react-resizable-panels` v2 固定**: v4 で API 大幅改名 (Group / Separator)、安易な upgrade 禁止

### v62 維持
- **sticky 検索バー** (Apple 方式 backdrop-filter): SPA path のみで使用、workspace mode は触らない
- **発光系 .panel-card 等**: 新規 card 追加前に必ず `glow_elevation_postmortem.md` + `css_specificity_gotchas.md` 読む
- **JudgmentRow** (= ws-judgment-row class 付き): v2 archived path と workspace mode 両方で使用
- **`pre_edit_design_tokens.sh` / `pre_edit_trust_cliff.sh` hook**: frontend/ + frontend-next/ 両 path 検査対象
