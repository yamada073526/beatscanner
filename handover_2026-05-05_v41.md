- 現在 yfinance fallback (10 ETF proxies) で 54 件配信中

**次セッション検証手順** (1-2 分):
```bash
# Step 1: API レスポンスを確認
curl -s "https://beatscanner-production.up.railway.app/api/macro-news?_=$(date +%s)" | python3 -c "
import json, sys
d = json.load(sys.stdin)
items = d.get('items', [])
meta = d.get('_meta', {})
print(f'{len(items)} items, _meta: {meta}')
av = [x for x in items if x.get('_kind') == 'av']
print(f'AV items: {len(av)}')
for x in av[:10]:
    print(f'  ({x.get(\"source\",\"?\")}) {x[\"title\"][:70]}')
"

# Step 2: 重要キーワード hit 確認
curl -s "https://beatscanner-production.up.railway.app/api/macro-news?_=$(date +%s)" | python3 -c "
import json, sys
d = json.load(sys.stdin)
items = d.get('items', [])
for kw in ['jpmorgan', 'morgan stanley', 'goldman', 'raises target', 'year-end target', 'iran', 'attack']:
    hits = [x for x in items if kw in (x['title']+' '+(x.get('summary') or '')).lower()]
    if hits:
        print(f'{kw}: {len(hits)} 件')
        for h in hits[:2]:
            print(f'  [{h[\"importance\"]}|{h[\"category\"]}] ({h.get(\"_kind\",\"?\")}) {h[\"title\"][:70]}')
"
```

**判定**:
- ✅ AV 10+ 件 + WSJ/Reuters/Bloomberg ソースあり: 成功 → 状況次第で完了
- ⚠️ AV 機能するが対象記事なし: タイミング依存、1-2 日後に再確認
- ❌ AV 0 件: 引き続き quota 問題 → §17-6 の Plan B (Claude Haiku L3 / Marketaux) 検討

**根本解**: FMP 有料化（後述）または Trading Economics で構造的に解決見込み。

## 5. 既知の制約・未解決問題

### 5-1. FMP free 枠の制約
- `/general-news`: 429 (handover v40 §17)
- `/economic-calendar`: 429
- `/quote` レート制限: 一部 ticker で fallback 発動
- 解決: FMP 有料化、または Trading Economics ($25/月) 等の代替ソース

### 5-2. AlphaVantage NEWS_SENTIMENT
- 25 req/日 制約 (3h cache でも複数デプロイ + ユーザー閲覧で消費しがち)
- 当日 quota 消費後は yfinance fallback に依存
- 動作確認は handover §17-5 手順で

### 5-3. 翻訳 latency
- 初回 ~10s (キャッシュなし時)
- リピーターは localStorage hit で 0s (本セッション改善済)
- 中期施策 SSE ストリーミングで初回も TTFT 0.5s 化見込み

### 5-4. 経済カレンダーの精度
- 静的 fallback は米国標準 recurring schedule に基づく**推定日**
- FOMC/BOJ/ECB は 2026 年公表値準拠だが小ズレ可能性
- FMP 有料化で完全解決
