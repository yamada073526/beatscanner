## 0. エグゼクティブサマリー（結論先出し）

- **ニュースを「独立した第4の柱」に昇格させるのは筋が悪い。** 競合を見ると、ニュースは「①速報を看板にする組（資本ゲーム）」と「②銘柄ページにぶら下げる衛生要因の組」に二分され、**個人開発が勝てる中間がない**。
- **ただし「文脈接着型ニュース」は競合の明確な空白地帯。** 全社が「ニュースを"並べる"」止まりで、**「このニュースは決算5条件のどれに効くか／買い場の質をどう変えるか」を自動で繋ぐ**プレイヤーがいない。ここは BeatScanner の本丸（解釈の自動化）とドンピシャ。
- **日本勢では moomoo が最大の脅威**（米国株の日本語ニュース・速報が無料で充実）。だが moomoo は「生データ＋コミュニティ」、BeatScanner は「答え（買い/見送り）を出す」で **土俵が違う**。

## 1. ニュースの2分類フレーム（競合の実態）

| タイプ | 代表 | ニュースの位置 | 勝ち筋 |
|---|---|---|---|
| 看板型 | Bloomberg / Benzinga / 株探 / moomoo | 速報・独自記事が商品 | リアルタイム配信網 or 記事を書く人員 = **資本/人員ゲーム** |
| 衛生要因型 | Yahoo / TradingView / Finviz / Koyfin / 証券各社 | 銘柄ページに時系列で羅列 | 「無いと不安」だから置くだけ。**差別化ゼロ** |
| 空白地帯 | （該当なし） | ニュース×解釈の自動接着 | ← **BeatScanner の狙い目** |

## 2. 4機能の競合ポジショニング（誰が看板にしているか）

- **ニュース看板**: Bloomberg, Benzinga（速報）, Seeking Alpha（分析記事）, 株探（決算速報記事）, moomoo（日本語速報）
- **決算看板**: Seeking Alpha（transcript/estimate/Quant）, 株探（"好決算"記事）, マネックス銘柄スカウター（過去業績ビジュアル）
- **チャート看板**: TradingView（業界最強）, Webull/moomoo（高機能テクニカル）
- **スクリーナー看板**: Finviz（定番）, TradingView, Koyfin（機関級データを安く）

→ **示唆**: BeatScanner の既存3機能のうち、**チャートとスクリーナーは"看板強者"がひしめく**（深追いは消耗戦）。**決算の"判定・解釈"は強者が手薄** ＝ ここが主戦場として正しい。

## 3. 主要競合プロファイル（ニュース観点を厚めに）⚠️ 価格要検証

### グローバル
- **Seeking Alpha** — リテール最強格。看板=Quant Rating(A–F)＋クラウドソース分析記事。ニュースは速報＋記事の二段。決算=transcript/estimate充実。料金 Premium ~$239/年（要検証）。
- **TradingView** — チャート看板。ニュースはアグリゲート（充実だが補助）。スクリーナーも強い。ソーシャル＋Pine Script が堀。
- **Finviz** — スクリーナー＆ヒートマップ看板。ニュースは銘柄ページ羅列。Elite課金。
- **Koyfin** — 「Bloombergの廉価版」。機関級データ・財務・estimateを安く。ニュースは補助。
- **Benzinga** — トレーダー向け速報看板（Squawk音声）。Benzinga Pro が高価。
- **Yahoo Finance** — 万人向けポータル。ニュース量は最大級だが文脈なし。
- **Bloomberg / Morningstar / simplywall.st / Webull / Robinhood / Fey / Stock Events** — それぞれ機関向け／長期投資（★レーティング）／ビジュアル分析／取引本体に情報付随／通知アプリ、と棲み分け。

### 日本（米国株情報）
- **moomoo証券(日本)** — ⚠️ **最大の直接競合**。無料で米国株の日本語ニュース・速報・決算カレンダー・高機能チャート・コミュニティ。情報量では真っ向勝負しても勝てない。
- **マネックス証券「銘柄スカウター 米国株」** — 過去業績のビジュアル分析が高評価。決算分析の直接競合に近い。
- **楽天証券 / SBI証券** — 口座付随の日経・ロイター配信＋スクリーナー。情報は取引のおまけ。
- **株探（kabutan）** — 日本株で「決算速報を記事化」する文化が超強い。**米国株でこれをやる所が手薄** ＝ 隙間の傍証。
- **みんかぶ / 日経 / Bloomberg日本語** — メディア型。

## 4. BeatScanner への示唆（依頼の4問に回答）

1. **第4の柱に昇格すべきか** → **No寄り。** 看板型は資本で負け、衛生型は差別化ゼロ。「タブを増やす」＝「読み手に負担」で原則に逆行。
2. **足すなら最も効く形** → **文脈接着型（ニュースを決算・株価の"なぜ"に紐づけ）が競合空白地帯。** 次点で監視代替型（保有/WL銘柄に重大ニュース→通知、Stock Events＋じっちゃま解釈）。AI記事配信型は既存ロードマップだが工数大。
3. **日本勢への立ち位置** → moomooに情報量で勝てないが、**「解釈・判断の自動化（買い/見送りの答え）」「日本語×米国株×じっちゃまプロトコル」は隙間**。moomoo=データ、BeatScanner=答え。
4. **他3機能の競争力** → 決算判定=独自で強い／チャート=強者に機能劣後（深追い不要）／スクリーナー=条件数では劣るが **じっちゃま較正プリセットは差別化余地**。

## 5. deep-research で裏取り・拡充すべき点（draft の穴）

- 各社の **正確な料金・無料/有料の線引き**（記憶ベースで不正確）
- moomoo日本・マネックス銘柄スカウターの **ニュース/決算機能の最新実態**（直接競合なので重要）
- 「ニュース×解釈の自動接着」を **やっている新興がいないか** の確認（空白地帯の検証 ＝ ここが崩れると結論が変わる）
- 各社の **ニュースAI要約・翻訳** の有無（後発の差別化が消えてないか）
- 株探US等、**米国株決算の記事化** を日本語でやる競合の有無
