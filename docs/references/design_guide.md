# デザインガイドライン

## 設計哲学
「2秒でメンタルモデルが構築できる」を最優先とする。

## カラー定義
| 用途 | カラーコード |
|------|-------------|
| PASS（合格） | #22c55e（緑） |
| FAIL（不合格） | #ef4444（赤） |
| 中立・背景 | グレー系 |
| 強調（5%以上変動） | 🔴🔵で視覚的に強調 |

## レイアウト原則
- 判定結果（PASS/FAIL）は画面上部・大きく表示
- 数値根拠は画面下部にカード形式で並列表示
- 余白を広くとりノイズを排除する
- 数値は大きめ・太字で視認性を確保

## テーブル形式
- 並列比較はテーブル形式を優先
- 5%以上の変動は🔴（下落）🔵（上昇）で強調表示
