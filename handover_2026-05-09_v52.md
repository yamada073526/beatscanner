## 3. 🟡 重要知見 (v52 セッション固有)

### 3-1. 「発光演出」の合意: 撤廃ではなく IntersectionObserver で精緻化

**3 体レビュー結果**:
- UI/UX agent: **撤廃推奨** (Aman は「動かない」、Linear 不採用)
- Web 設計 agent: **継続 + IO 化 + token 化** (`data-surface` 属性)
- Web 開発 agent: **継続 + IO 化 + class 統合** (`.surface-card`)

→ **2:1 で「リファクタして残す」優勢**。ユーザー希望「マイナスでなければ残してほしい」とも整合。

実装方針:
- 旧 `querySelectorAll('.panel-card').forEach` + scroll listener (毎 frame layout thrash) → IntersectionObserver
- **1 枚だけ active** (中央 ±25% に最も近いカード) で視線集約 (Aman 単一スポットライト原則)
- `.panel-card` / `.bs-panel` / `.surface-card` 統合 → Portfolio/Watchlist も対象
- `prefers-reduced-motion` で hook 自体を作らない

### 3-2. ウォッチリスト統合は **案 A (上下分割)** 全員一致

- A: 上下分割 (チャート上 / リスト下) ← **採用**
- B: 左右 2 ペイン (Bloomberg)
- C: peek モーダル (Linear)
- D: 現状維持

PC は @container query で 2 ペインに自動切替も検討可。Week 2 で着手。

### 3-3. デザイントークン命名規約 (永続)

- spacing: `--space-1 / 2 / 3 / 4 / 6 / 8 / 12 / 16` (4-base scale, Stripe Dashboard 流)
- radius: `--radius-xs / sm / md / lg / xl / pill` (Vercel Geist 流)
- shadow: `--shadow-1 / 2 / 3 / 4` + `--shadow-glow-cyan` (Aman 4 階層)
- motion: `--ease-out-expo / in-out-quad / out-cubic` (Linear 標準 cubic-bezier)
- duration: `--motion-fast (120ms) / base (200ms) / slow (360ms) / stage (600ms)`
- z-index: `--z-base / sticky / overlay / modal / toast`

### 3-4. KPI 大数字に Inter cv11 + opsz が劇的に効く

`font-feature-settings: "tnum","cv11","ss01"` + `font-variation-settings: "opsz" 56` で Bloomberg Terminal 級の数字重み。

### 3-5. NYSE 市場状態判定は frontend のみで実装可

`Date.toLocaleString('en-US', {timeZone: 'America/New_York'})` で ET 時刻を取得 → 平日 09:30-16:00 で REGULAR 等の状態判定。
祝日カレンダーは省略 (将来 backend `/api/market-calendar` で厳密化)。

### 3-6. Vite manualChunks 拡張で初期バンドル -68%

```js

  'react-vendor': ['react', 'react-dom'],
  'supabase': ['@supabase/supabase-js'],
  'charts': ['lightweight-charts', 'recharts'],   // 新規 (-200KB)
  'dnd': ['@dnd-kit/core', '@dnd-kit/sortable', '@dnd-kit/utilities'],  // 新規
  'markdown': ['react-markdown'],                  // 新規 (-118KB)
}
```
