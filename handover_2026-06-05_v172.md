真因 = FMP `/earnings-calendar` (range fetch) の ~4000 件上限 (決算ピーク期 7-8月は 30日でも超過 → 日付後半の NVDA 8/26 等が欠落)。 per-ticker `/stable/earnings` に置換して完全回避。

`main.py` の `_fetch_range` + `_fetch_range_chunked` + `rows_future/past` gather + `by_sym` ループ (`get_holdings_meta` 内、 約 1576-1638) を以下に置換 (コメント/date 変数の Edit `date_from_past`/`date_to` 削除は前セッションで working tree 適用済か要確認):

```python
    async def _fetch_ticker_earnings(sym: str) -> tuple[str, list[dict]]:
        _ckey = f"tkr:{sym}"
        _cl = _EARNINGS_RANGE_CACHE.get(_ckey)
        if _cl and now - _cl["ts"] < _EARNINGS_RANGE_TTL:
            return sym, _cl["data"]
        try:
            _client = FMPClient(api_key=api_key)
            _data = await _client.earnings_surprises(sym, limit=12) or []
        except Exception:
            _data = []
        _EARNINGS_RANGE_CACHE[_ckey] = {"data": _data, "ts": now}
        return sym, _data

    _ticker_results = await asyncio.gather(*[_fetch_ticker_earnings(s) for s in syms])

    by_sym_next: dict[str, str] = {}
    by_sym_last: dict[str, dict] = {}
    for sym, ticker_rows in _ticker_results:
        for r in ticker_rows:
            if not isinstance(r, dict):
                continue
            d = r.get("date")
            if not d:
                continue
            try:
                d_iso = str(d)[:10]
                if d_iso >= date_from:
                    cur = by_sym_next.get(sym)
                    if cur is None or d_iso < cur:
                        by_sym_next[sym] = d_iso
                else:
                    cur = by_sym_last.get(sym)
                    if cur is None or d_iso > cur.get("date", ""):
                        by_sym_last[sym] = {
                            "date": d_iso,
                            "eps_actual": r.get("epsActual") or r.get("eps"),
                            "eps_estimated": r.get("epsEstimated") or r.get("estimatedEps"),
                        }
            except Exception:
                continue
```
