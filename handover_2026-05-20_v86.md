## 2. v86 で確立した永続教訓 (memory 起票済 + 起票候補)

### 起票済 (`memory/feedback_pge_loop_pitfalls.md`)

PGE 3 体ループの 4 落とし穴:
1. worktree 非累積 (各 sprint が main から独立 branch、 sprint 完了ごとに main consolidate 必要)
2. Evaluator L3 selector hallucination (React component 名 = className という暗黙前提)
3. ESM top-level return SyntaxError (try 直下 return も module top level 不可)
4. `getAnimations().forEach((a) => a.finish())` の InvalidStateError (infinite animation 対策)

### 起票候補 (handover v86 → next session)

#### A. Vision API noise pattern (`feedback_vision_api_noise.md`)

- single 測定 ±4pt noise
- LLM 評価は self-contradiction あり (「強く」 と「subtle」 を交互に提案)
- 静止 PNG では motion 系を検知困難 (LIVE pulse / glow halo の score 変化 +0)
- 真の差検出には **3 run mean** または **build 別 5-10 run sample** が必要

#### B. Chart hybrid 4 層防御継承 (`feedback_chart_overlay_safety.md` 更新)

Recharts custom shape (Bar.shape) でも 4 層防御適用:
- ErrorBoundary (StockChartErrorBoundary 継承)
- conditional render (chartStyle === 'candle' branch)
- Number.isFinite guard (OHLC 全 field)
- isAnimationActive=false (CandleShape は静的描画)
