## 2. 🔴 次セッション最優先 — Phase 0 バグ修正 (Vite 上で先行)

道A 移行は時間かかるが、以下 2 バグはユーザー dogfood 続行 + 信頼性のため**先に Vite で修正・リリース**:

### Bug 1: LP attention chips 発光 (HomeTab.jsx)
- **症状**: 3 チップのうち 1 個ランダム発光、3 個同時に発光しない
- **根本原因 (推定)**: `useArrivalSpotlight` (frontend/src/hooks/useArrivalSpotlight.js:51-53) の leaf filter が「最も画面中央に近い 1 枚」のみ active にする仕様
- **修正方針**: HomeTab の attention 3 チップ用に **section 単位の単一 IntersectionObserver** + 3 チップに staggered (0/100/200ms) で `is-arriving` を付与
- **ファイル**: `frontend/src/components/HomeTab.jsx`
- **CSS**: 既存 `.is-arriving` 再利用、新規 CSS 不要

### Bug 2: ApiKeyBanner 残置 (App.jsx:766)
- **症状**: 判定タブ / 決算タブ / チャートタブ上部に「⚡ FMP API キー...」banner 残置
- **修正案**:
  ```jsx
  {!showLP && !isProUser && activeTab === 'home' && (
    <ApiKeyBanner ... />
  )}
  ```
- **ファイル**: `frontend/src/App.jsx`

両方 1 時間以内に修正、Vite で `railway up` → ユーザーが本番確認 → Phase 1 着手。
