- `SPARKLINE_PERIOD_OPTIONS` export を **削除しない** (IndicesView/PeriodChipBar が使用)

### WorkspaceScreenerModal 着地後の禁止事項
- focus trap (`querySelectorAll('a[href], button:not([disabled])...')`) を **削除しない**
- `Workspace.jsx` の **lazy + Suspense** を **static import に戻さない** (bundle 肥大)
- `App.jsx` `detailContext.onUpgrade(featureName)` の **default fallback を削除しない** (backward compat、 旧 callsite が引数なしで呼ぶ)
- 旧 SPA mode の drawer button (App.jsx L2295) からの screener access は **regression なし**、 削除しない

### changePct fraction 変換 (App.jsx)
- `px?.change_pct != null ? px.change_pct / 100 : null` の **`/ 100` を削除しない** (backend は percent unit、 frontend は fraction 想定)
