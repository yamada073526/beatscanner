## 3. 触ると危険な箇所 (v104 + v105 教訓)

### v104 から継承
- sticky 検索バー / 発光系 / VITE_ARG/ENV / Hallucination Guard / FMP /stable
- ReadingMode 全面 Overlay / Pane 3 章 2 ChapterTabs 二重表示防止 `!isV3 &&`
- App.jsx lucide-react import は v105 で追加完了 (theme toggle + mobile bottom tab + その他 5 component)

### v105 新規教訓
- **vision-eval は anonymous user で demo rate limit (3 req/IP/day) に hit**: PDCA 自動化を本番で回すには auth bypass token か OAuth flow 必要。 spawn_task chip で release 後の課題化済
- **snap-pdca-loop v1 の wait timing 不足**: 8s selector wait + 4s ticker click では analyze API + Pane 3 mount まで届かない。 v2 で 25s + 88s timeout + URL ticker detection に拡張
- **ch2tab URL state は history.replaceState で navigation 起こさない**: pushState だと back button で複雑 / SPA Router と conflict。 default param 削除で URL クリーン維持
- **spawn_task chip は同 worktree で並走実行されうる**: 本来別 worktree で動く設計だが、 自律実行中に modify されることがある。 git diff で確認 + selective commit で安全に分離
- **lucide-react import を App.jsx に追加すると bundle main entry size +1.5KB**: 343→343.36 KB の minor 増加、 既存 framer-motion / supabase 並みで許容範囲
