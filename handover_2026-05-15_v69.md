### 必須回避 (致命的、6 体一致)

- **単純 P/L 差分は禁止** (cash flow を儲けと誤認、$10k 入金で「+$10k 儲け」と誤表示 → Trust Cliff 級バグ)
- **Modified Dietz**: `(期末 − 期初 − 純 cash flow) / (期初 + 加重 cash flow)` が retail 業界標準 (Sharesight / Personal Capital / Snowball)

- **partial period**: 期間中の corporate action (split, spinoff) + 期間中の取得銘柄を Dietz 分母に加重平均で含める
- **数値は AI 触らせない** (Anthropic): hallucination 即 Trust Cliff、Claude は表示テキスト生成のみ

## 5. 触ると危険な箇所 (CLAUDE.md + v68/v69 追加)

### v69 追加
- **`_safe_float` L339 (2-arg) vs L2054 (1-arg) の名前衝突**: 後者 (`_safe_eps_float` に rename 済) が 1-arg + 0.0→None の特殊版。**新たに 2-arg `_safe_float(x, ndigits)` を使う場合は L2054 (=現 L2054 周辺) で再 shadow されないか確認**。EPS context は `_safe_eps_float` に集約済。
- **`WorkspaceShell の contain: layout`** が `position: fixed` の containing block を pane に固定する仕様。新規 modal を作る場合は **必ず createPortal で document.body 直下に描画** (TransactionEntryModal / TransactionHistoryModal / PortfolioJudgmentDetailModal が pattern)。
- **`useTransactions` window event** (`bs:transactions:changed`) は同 user 内の複数 instance 同期に必須。新 hook 追加時は同 event を購読すること。
- **`setSelectedAccountId` は filterTicker を null reset する** (v68 §2 #6 dogfood 規約)。account 切替 + ticker filter を同時に行う場合は **setSelectedAccountId → setFilterTicker の順序が必須**。
- **`switchAppTab('indices')` 経由で SPA + workspaceStore 両方の activeTab を同期** (Cmd+K 経由の口座切替が workspace ホームタブから indices タブに切替らない bug の根本対策)。
- **Cmd+K palette の Enter / Esc は IME composition 中 (isComposing / keyCode === 229) は skip**。macOS の Cmd+K (カタカナ変換) と衝突しない。
- **Trust Cliff 規律**: `_INSIGHTS_SYSTEM_PROMPT` と `_sanitize_insights_text` に「じっちゃま」「広瀬」関連を **二重防御** で実装済。新たに Claude が出力する UI テキストを追加する場合は **必ず両方に禁止語登録**。
- **AI 数値生成は厳禁**: 含み損益 / 期間 P/L / 評価額等の数値は backend 決定論で計算、Claude は表示テキスト生成のみ (hallucination が Trust Cliff 直撃)。
