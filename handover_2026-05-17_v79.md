## 3. 触ると危険な箇所 (v79 新規 + 継承)

### v79 新規確定原則

**Chip tone `'elite'` (gold #d4af37) は extreme value 専用 (handover v79、 chip_primitive_canonical.md 更新)**:
- RS percentile ≥ 95 / ≤ 5 で発火 (現状 75/25 の更に外側)
- 投資業界色ルール (緑=上昇 / 赤=下落 / amber=警告 / cyan=ブランド / gold=希少 elite) に 1 hue 追加
- 通常の gain/loss tone と被らない別意味として 1 秒識別
- 多用すると希少性希薄化 → Phase 1 では RS extreme のみ、 他指標は Phase 2 で再評価

**Chip `data-fresh` 属性 + gain tone 濃化 (handover v79)**:
- `days_ago ≤ 14` 等の「新鮮シグナル」 で gain tone を 8%→14% / 30%→45% / text 95% に濃化
- pulse animation は不採用 (#1「2 秒視認」 違反、 「狼少年」 化リスク)
- DMA cross のみ Phase 1 で導入、 他 signal は Phase 2 で再評価

**「数値 + 1 メタ」 chip ルール (subagent 共通 verdict)**:
- chip 1 個に込められるメタ情報は数値 + 1 個まで (例: 「✦ ゴールデンクロス 7 日前」 / 「上位 2% · RS +178%」)
- それ以上は tooltip / popover に逃がす (Bloomberg F8 detail 思想)
- Cup-Handle breakout_pending = 3 情報が限界 (「☕ + status + days_to_breakout」)
